Actserv Full Stack Developer Assessment

Dynamic Form Builder and Submission Management System

This platform demonstrates a Full-Stack, component-driven solution featuring an optimized Django/DRF backend and a responsive React frontend. The backend is architected around the Entity-Attribute-Value (EAV) pattern for dynamic form schema creation, while integrating asynchronous Celery processing for scalable admin notifications and secure file upload handling. The system provides separate, feature-rich Admin and Client APIs, proving the ability to deliver both complex data management tools and simple public user interfaces.
